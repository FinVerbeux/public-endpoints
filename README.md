# Public Web Endpoints

Collection d’endpoints **publics et non authentifiés** découverts
via l’inspection réseau (Network tab) de sites officiels.

## Objectif
- Centraliser des endpoints JSON publics
- Faciliter la recherche et l’analyse open‑data
- Éviter de re‑découvrir les mêmes APIs

## Méthodologie
- Inspecteur navigateur → Network → Fetch/XHR
- Aucun contournement de sécurité
- Aucun endpoint privé ou authentifié

## Sources
- Sites publics
- APIs exposées volontairement par les plateformes

## ⚠️ Disclaimer
Ce dépôt ne contient **aucune donnée privée** et n’encourage
aucune utilisation abusive.
